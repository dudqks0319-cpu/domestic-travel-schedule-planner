# TripMate Mobile (Step 1-2)

This package contains the corrected Step 1-2 frontend implementation in Expo Router form.

## Run

1. `npm install`
2. `npm run start`

## Notes

- Uses local AsyncStorage for temporary auth/profile state.
- Password is not passed through route params.
- Profile setup progress is displayed as 1/5..5/5 inside setup flow.
